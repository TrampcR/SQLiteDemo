#SQLiteDemo

### 1.利用SharedPreferences实现是否开启引导界面
第一次打开应用会出现引导界面，引导界面运用了ViewPager实现左右滑动。进入主界面后，退出应用，第二次打开应用时不会再进入引导界面。

### 2.利用SQLite保存数据，实现简单的注册和登录功能
第一次打开应用时，会跳到注册、登录界面，进入注册界面，注册的数据都会存入到数据库，注册完毕，会直接进入主界面，并显示用户名，这时数据库中该用户名的状态为“登录中”。如果用户点击了退出键，下次打开应用时会进入登录界面；如果用户没有点击退出键，下次打开应用时会直接进入主界面。
